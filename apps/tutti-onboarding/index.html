先把你的 Agent 接进来

目前仅支持 Claude Code 和 Codex。更多 Agent 能力正在开发中 💻

绑定后就可以在 Tutti 里使用你的 Agent 啦。无需终端，上手简单。不同 Agent 间交互体验一致。

⚡

看看 Agent 怎么协作

对话中打个 @，让 Agent 获取彼此的会话、文件、任务，带着上下文工作；也可以 @ 应用，让 Agent 使用。

🎯 你说目标，Agent 拆解执行

说你想做什么就行。接着再委派 Agent 自动拆成子任务，指定合适的 Agent 去执行。

📡 全局任务管控

Agent 的消息、运行状态、异常，不用你盯着，也不会漏掉。

✨

试试内置应用

内置一系列生产力应用，开箱即用。复用已有的 Agent 订阅 i可使用的应用能力取决于你已有的 Agent / 模型订阅

Agent 可以直接 @ 应用完成任务，也能拿应用的产物继续工作。不需要你手动搬运。
