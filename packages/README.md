# Packages

Shared packages for `tutti-apps`.

Create a package here when code is reused by more than one app or needs an
app-independent contract.

Recommended package examples:

- `@tutti-apps/config`: shared TypeScript, Tailwind, or lint config.
- `@tutti-apps/github`: GitHub REST client and API types.
- `@tutti-apps/trend-classification`: reusable taxonomy and scoring logic.
- `@tutti-apps/ui`: shared UI primitives only if multiple apps need the same
  components.

Read `AGENTS.md` before editing or adding packages.
